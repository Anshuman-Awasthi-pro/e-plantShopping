# e-plantShopping

e-plantShopping is a React-based shopping cart application for an online plant store called Paradise Nursery.

This application allows users to:
- Browse a variety of houseplants organized by categories
- Add plants to a shopping cart
- View cart items and total cost
- Increase, decrease, or remove items from the cart
- Navigate between the landing page, product listing page, and shopping cart page

The project is built using React and Redux Toolkit and is deployed as a public web application.
